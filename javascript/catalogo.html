Reto#4

El programa debe permitir al usuario seleccionar un nombre de una lista, un apellido de otra lista y una actividad de una tercera lista. Si la selección del nombre, apellido y actividad coincide con la respuesta correcta, el programa mostrará un mensaje que diga "La respuesta es correcta". En caso contrario, mostrará un mensaje que indique "La respuesta es incorrecta".

Reto#5

programa que muestre las cuatro figuras geométricas básicas (círculo, triángulo, rectángulo, cuadrado) mediante tarjetas. Cada tarjeta tendrá dos botones: "Perímetro" y "Área". Cuando el usuario haga clic en uno de estos botones, se desplegará una ventana modal que permitirá al usuario ingresar los datos necesarios para calcular la operación correspondiente a la figura seleccionada. La ventana modal incluirá un botón "Calcular" para mostrar el resultado de la operación y un botón "Cerrar" para cerrar la ventana modal.
