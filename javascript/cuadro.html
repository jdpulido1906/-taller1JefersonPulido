Eventos JavaScript

Evento Ejemplo de aplicación Demostración
click Se activa cuando se hace clic en un elemento
mouseover para detectar cuando el cursor se coloca sobre la imagen y luego aplicamos la rotación
focus Cambiar el estilo de un campo de texto cuando recibe el enfoque.Esta se activa cuando le das click a la imagen
hashchange Se activa cuando cambia el fragmento de la URL (hash) Ir a la sección 1
keydown Se activa cuando se presiona una tecla

contextmenu Se activa cuando se hace clic con el botón derecho del mouse para abrir el menú contextual. Haz clic derecho aquí
keyup Se activa cuando se suelta una tecla
mouseup Se activa cuando se suelta un botón del mouse
dragstart Se activa cuando se inicia el arrastre de un elemento
beforeprint Se activa antes de iniciar el proceso de impresión
Sección 1 JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Se utiliza principalmente para agregar interactividad y dinamismo a las páginas web. JavaScript permite a los desarrolladores crear scripts que se ejecutan en el navegador del cliente, lo que brinda una amplia gama de capacidades para interactuar con el contenido de la página, manipular elementos HTML, realizar validaciones de formularios, realizar peticiones al servidor, entre otras tareas.
